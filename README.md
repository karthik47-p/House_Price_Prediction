# House Price Prediction

**Author:** P.N.Karthik

## Project Overview
This project predicts house prices using machine learning.  
It includes: 
- Data Analysis (EDA)  
- Data preprocessing (handling missing values, encoding)  
- Regression model training  
- Predictions on test data  

## Libraries Used
- Python 3  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn  
- Jupyter Notebook
